Node JS

javascript no backend (server-side platform to OS X, Microsoft Windows, and Linux).

Ao instalar o Node no sistema operacional Windows este programa é registrado nas variaveis de ambiente, ficando disponível via terminal (prompt) do SO ou das IDEs.

Após a instalação pode-se verificar a versão no prompt com o comando: node -v

Caracteristicas: assíncrono, rápido, enumeras requisições e não utiliza buffer.

Ideal para I/O, streaming de dados, real-time, API JSon nativa e SPA.
